# RemoveBG

A simple web app to remove image backgrounds using Hugging Face Transformers.  

**Features:**
- Upload an image and remove its background instantly.
- Shows a loading state while the image is being processed.
- Fully client-side using `@huggingface/transformers`.

**How to use:**
1. Click the **upload button** and select an image.
2. Wait while the background is processed (loading message will appear).
3. See the resulting image with the background removed.
